# Your Music State

### Descripción general

Página Web que muestra informácion de tu gusto musical usando Spotify.
Muestra tus canciones y artistas más escuchados, tus splaylists guardadas, e incluse te deja buscar albums.

### Tecnologías Usadas:

- HTML
- CSS
- JavaScript
- DOM
- Figma
- Imágenes
- OOp
- Webpack
- APIs
- React
